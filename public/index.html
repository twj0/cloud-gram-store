如何使用 WebDAV

Windows 文件资源管理器
  1. 打开文件资源管理器
  2. 右键点击"此电脑" → "映射网络驱动器"
  3. 输入 WebDAV 地址
  4. 输入用户名和密码
macOS Finder
  1. 打开 Finder
  2. 按 Cmd+K 或选择"前往" → "连接服务器"
  3. 输入 WebDAV 地址
  4. 输入用户名和密码
第三方客户端

推荐使用：WinSCP、Cyberduck、rclone 等支持 WebDAV 的客户端
